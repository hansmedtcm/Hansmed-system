# Coder Instruction — Rename "AI Diagnosis" → "TCM Wellness Assessment"

**Version:** 1.0
**Date:** 2026-04-21
**Decision authority:** Product owner (HansMed)
**Compliance rationale:** Malaysia MDA 2012 (non-diagnostic framing), PDPA 2010 (accurate notice), Claude Usage Policy (no medical-diagnosis framing for AI output)
**Scope for this task:** Frontend strings only (EN/BM/ZH). Backend API, DB columns, and URLs are **out of scope** for this sprint.

---

## 1. The canonical naming

**Feature name (locked):** `TCM Wellness Assessment`

**Short form / nav label:** `Wellness Assessment`

**Icon tooltip:** `TCM Wellness Assessment`

**Do NOT mention "AI" anywhere in the UI for this feature.** AI involvement is disclosed in Privacy Notice § 5 only. This is a deliberate product decision.

### Trilingual lock

| Language | Full name | Short / nav |
|---|---|---|
| English | TCM Wellness Assessment | Wellness Assessment |
| Bahasa Malaysia | Penilaian Kesihatan TCM | Penilaian Kesihatan |
| 中文 (简体) | 中医健康评估 | 健康评估 |

Use these strings exactly. Do not paraphrase.

Apply to frontend only (`v2/**/*.html`, `v2/**/*.js`). Preserve case where possible.

| `View diagnosis` | `View assessment` | |
| `Your diagnosis` | `Your wellness assessment` | |

### Do NOT replace these

- `Practitioner diagnosis` — keep if it refers to a licensed practitioner's professional assessment (rare; but check context)
- `Differential diagnosis` — only appears in practitioner-side clinical notes; leave
- `Self-diagnosis` — only appears in warning copy ("do not self-diagnose"); leave

### Manual review required

- `Diagnosis` appearing alone — check context. If referring to AI feature → rename. If medical context warning → leave.
- `Diagnose` (verb) — usually replace with `Assess`.

## 4. Required visual touches

### 4.1 Page header on the feature page

**Before:**
```html
<h1>AI Diagnosis</h1>
<p class="subtitle">Get your diagnosis powered by AI</p>
```

**After:**
```html
<h1 data-i18n="feature.assessment.title">TCM Wellness Assessment</h1>
<p class="subtitle" data-i18n="feature.assessment.subtitle">
  Understand your body through the lens of traditional Chinese medicine.
</p>
```

### 4.2 Required disclaimer (D1 from compliance doc)

Must appear **above** any output block:

```html
<div class="disclaimer disclaimer--assessment" role="note">
  <strong>Important:</strong>
  This wellness assessment is generated by software for educational purposes only.
  It is not a medical diagnosis, does not replace consultation with a registered
  healthcare practitioner, and should not be used to make treatment decisions.
  If you have symptoms that concern you, please consult a qualified practitioner.
</div>
```

Minimum font size 14px. Must not be collapsed by default. Must survive language toggle (BM and ZH versions in the compliance copy doc).

### 4.3 Nav label

**Before:**
```html
<a href="#ai-diagnosis">AI Diagnosis</a>
```

**After:**
```html
<a href="#ai-diagnosis" data-i18n="nav.assessment">Wellness Assessment</a>
```

Note: anchor/URL stays `#ai-diagnosis` for now — URL rename is out of scope.
